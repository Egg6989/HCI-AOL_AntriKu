"HCI-AOL_AntriKu" 

Meskipun sejumlah revisi telah berhasil diimplementasikan berdasarkan heuristic evaluation, beberapa fitur masih memiliki batasan karena pengembangan saat ini masih berfokus pada prototipe desain. Oleh karena itu, beberapa fungsi yang memerlukan integrasi backend, database, atau logika sistem yang lebih kompleks belum dapat direalisasikan sepenuhnya. Contohnya seperti validasi input dan pesan kesalahan, fungsionalitas password toggle yang belum didukung oleh logika JavaScript secara penuh, serta sistem antrean real-time yang masih menggunakan data statis.

Meskipun itu, seluruh fitur tersebut telah dipertimbangkan dalam proses perancangan dan direpresentasikan melalui elemen antarmuka yang sesuai untuk menunjukkan perilaku sistem yang diharapkan.
